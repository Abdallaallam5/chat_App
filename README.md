# Real-Time Chat App

A simple real-time chat application built with Node.js, Express, and Socket.IO.

## Features
- Real-time messaging
- Multi-user support
- Broadcast messages
- Instant message delivery using WebSockets

## Tech Stack
- Node.js
- Express.js
- Socket.IO
- HTML
- CSS
- JavaScript

## Installation

```bash
git clone https://github.com/Abdallaallam5/chat_App.git
cd chat_App
npm install
npm start
